# Модель температуры при наплавке

Готовый React + Vite проект для GitHub Pages.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из этого проекта в репозиторий.
3. Убедись, что основная ветка называется `main`.
4. На GitHub открой `Settings` -> `Pages`.
5. В разделе `Build and deployment` выбери `Source: GitHub Actions`.
6. Сделай commit и push в ветку `main`.
7. GitHub сам соберёт сайт и опубликует его.

После публикации ссылка на сайт появится в разделе `Pages` и во вкладке `Actions`.
